# 4G (LTE / LTE-A)

Кратко
- All-IP сеть, OFDMA (downlink) и SC-FDMA (uplink), MIMO.
- Высокая пропускная способность, низкая задержка по сравнению с 3G.

Ключевые понятия
- Архитектура: eNodeB, EPC (MME, SGW, PGW)
- Протоколы: S1, X2; VoLTE для голоса в IP
- LTE-A: carrier aggregation, advanced MIMO
