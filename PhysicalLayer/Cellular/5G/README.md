# 5G (NR)

Кратко
- New Radio (NR), поддержка широкого набора сервисов: eMBB, URLLC, mMTC.
- gNB в RAN, 5GC с SBA (AMF/SMF/UPF и др.).

Ключевые понятия
- Частоты: sub-6 GHz и mmWave; massive MIMO и beamforming
- Network slicing, edge computing, SA vs NSA
- Низкая латентность и высокая плотность подключений

Вопросы для собеса
- Что такое network slicing и как оно реализуется?
- Объясните разницу между 5G SA и NSA.

Ресурсы
- https://www.3gpp.org/release-15
- 3GPP TS 38.x


NR, gNB, RAN/core раздельные (NG-RAN, 5GC - AMF/SMF/UPF), mmWave, massive MIMO, beamforming, network slicing, URLLC/eMBB/mMTC, SA vs NSA, service-based architecture.
