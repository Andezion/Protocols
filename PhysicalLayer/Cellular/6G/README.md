# 6G (research / prospective)

Кратко
- Сильноспекулятивно: терагерцы, интеграция связи и сенсинга, AI-native сети.
- Фокус на семантической коммуникации, энергоэффективности и экстремально низкой латентности.

Ключевые понятия
- Технологические направления: THz, reconfigurable surfaces, integrated sensing
- AI/ML в управлении сетью и оптимизации ресурсов
- Безопасность и приватность в новых сценариях

Вопросы для собеса
- Какие технические вызовы у терагерцового диапазона?
- Как AI может изменить управление радиоресурсами?
