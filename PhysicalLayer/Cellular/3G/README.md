# 3G (UMTS / WCDMA / HSPA)

Кратко
- Переход к пакетному трафику и улучшенной скорости передачи данных.
- Основы: WCDMA/UMTS, позже HSPA для ускорения скоростей.

Ключевые понятия
- Архитектура: Node B, RNC, core — смешанные CS/PS возможности
- HSPA uplink/downlink ускорения
- QoS, задержки, голос + пакетные данные

Вопросы для собеса
- Чем отличается WCDMA от OFDMA?
- Роль RNC и как она соотносится с eNodeB/gNB в LTE/5G.
