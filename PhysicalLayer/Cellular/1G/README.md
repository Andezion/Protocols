# 1G

1G это аналоговые стандарты сотовой связи. Их суть в том, что передача голоса производилась с помощью аналоговых сигналов, а не цифровых, как в современных сетях. Именно поэтому там был высокий уровень помех, разговоры можно было легко перехватить(ибо не было шифровки), сети быстро перегружались и связь работала внутри лишь 1 локалькой сети.
