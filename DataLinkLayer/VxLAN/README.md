# VxLAN (Virtual Extensible Local Area Network) 

VXLAN (Virtual Extensible Local Area Network) - это технология сетевой виртуализации, которая инкапсулирует кадры Ethernet (уровень L2) в пакеты UDP (уровень L3). Она позволяет создавать виртуальные логические сети поверх существующей физической IP-инфраструктуры, растягивая сети L2 на огромные расстояния.
